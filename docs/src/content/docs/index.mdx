## Features

<CardGrid stagger>
  <Card title="Define Test Cases" icon="document">
    Create test cases with input strings and expected JSON outputs in a simple JSON format.
  </Card>
  <Card title="Compare Models" icon="random">
    Test against multiple LLM models via OpenRouter and compare their performance side by side.
  </Card>
  <Card title="Measure Performance" icon="rocket">
    Track accuracy, latency percentiles (P50, P95, P99), and throughput (tokens/second).
  </Card>
  <Card title="Multiple Output Formats" icon="list-format">
    Generate reports in terminal, JSON for CI/CD, or self-contained HTML for sharing.
  </Card>
</CardGrid>
